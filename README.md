# Sanos y Salvos - Frontend

Aplicación cliente desarrollada en React 18, diseñada para la visualización y reporte de mascotas perdidas/encontradas.

## Tecnologías Principales
- React + Vite: Framework de frontend ultrarrápido.
- Tailwind CSS: Diseño moderno y responsivo.
- Leaflet: Mapas interactivos para geolocalización.
- Axios: Cliente HTTP para comunicación con el API Gateway.

## Instalación y Ejecución Local
1. Asegúrate de tener Node.js (v18+) instalado.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abre http://localhost:3000 en tu navegador.

## Pruebas y Linting
Para verificar la calidad del código:
```bash
npm run lint
```

## Build para Producción
```bash
npm run build
```
Los archivos optimizados se generarán en la carpeta dist/.
